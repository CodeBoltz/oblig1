Preperation

To prepare for this oblig I reread all the content of the previous classes. This helped me remember all of the things we have discussed and allowed me to use all html elements correctly. On top of that I read all the additional material provided by Carlos.

Approach

For this Oblig I took a detailed look at what I needed to complete. I made a list of all the content that would be on my website. After collecting all the pages, I started to think about the content and what HTML elements I need for each section. This step helped me a lot with getting a good overview of the project and what I need to fulfill.

HTML elements used

  • Headlines - <h1>, <h2> and <h3> for creating hierarchy between my headlines. All styled with differnt sizes
  • Images - <img> tag for all images displayed on the site.
  • Links - <a> tag for links to my social media, subpages or other resources.
  • Text - <p> tag used for all text on my site.
  • Navigation bar - <nav> tag used for my navigation bar, allowing the user to navigate through my site easily.
  • Groups - <div> tag used to group content together and add associated class.
  • List - <ul> & <li> tag used for creating list to showcase my content clearly (for example nav bar or this list).
